# Sistema de Orçamento Imobiliário

Projeto desenvolvido para a disciplina Algorithmic Thinking & Introduction to Object-Oriented Programming.

## Funcionalidades

- Cálculo de orçamento para:
  - Apartamento
  - Casa
  - Estúdio
- Exibição do valor mensal
- Exibição do valor do contrato
- Geração de arquivo CSV com 12 parcelas

## Tecnologias utilizadas

- Python
- Programação Orientada a Objetos
- Git e GitHub
